# staxt
A minimal static site generator using Handlebars, Babel, Sass

## install
```
npm i staxt
```

## Cli Usage
```
npx staxt init
```
Creates a new prject with the following folder structure
```
-src
  -assets
    -scss
      -main.scss
    -js
     -main.js
  -templates
    -includes
      -header.hbs
      -footer.hbs
-dist
  -assets
    -main.js
    -main.css
    -images
```
